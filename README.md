# internetMemeCreator
This is a free online meme maker that allows you to add custom resizable text to images. It operates in HTML5 canvas, so your images are created instantly on your own device. Most commonly, it can be used to add text captions to established memes.

## Contents

The template project is just an example Hello World that guide you through the use of React with the Expressjs framework to build a very simple web application that display the Hello World!!! Greeting

# Can I use the generator for more than just memes?
Yes! The Meme Creator is a flexible tool for many purposes. You can design many creative works including posters, banners, advertisements, and other custom graphics.
